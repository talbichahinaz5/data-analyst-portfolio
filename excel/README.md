# Tableau de bord Excel – Analyse des ventes

Tableau de bord Excel interactif réalisé à l’aide de tableaux croisés dynamiques et de graphiques,
basé sur le jeu de données *Superstore*.

Le tableau de bord comprend :
- L’évolution des ventes par mois
- Les ventes par catégorie de produits
- Le top 10 des sous-catégories les plus rentables

Le fichier Excel peut être transmis sur demande.
